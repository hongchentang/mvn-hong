教育创新人才培训计划项目培训需求表的通知

发布日期：2014-12-19 浏览次数：20

2014年3月7日，由教育部师范教育司领导和指导，全国中小学教师教育技术能力建设计划项目办公室组织开展的第三届全国中小学教师教育技术能 力建设计划应用成果评比活动完满结束。广东第二师范学院远程培训中心、广东省汕头职业技术学院现代教育技术中心被授予优秀组织管理奖。我省推 荐的来自广州、东莞、江门、韶关、汕尾、佛山、珠海、潮州、汕头、清远、梅州、中山、河源、阳江14个地市的42件教学设计方案获优秀作品奖。其 中《绿色的梦》等7件作品获一等奖；《我的绘本》等7件作品获二等奖；《项脊轩志》等13件作品获三等奖；《散步》等15件作品获优秀奖。

依据《关于举办第三届全国中小学教师教育技术能力建设计划应用成果展示活动的通知》的要求，2013年10月，广东省中小学教师教育技术能力建 设项目办公室（以下简称“省项目办”）组织开展全省优秀成果收集以及遴选工作。经过各地级市中小学教师教育技术能力培训执行机构的初选，和省 项目办组织专家评审，我省共推荐2个优秀组织管理单位、72件优秀作品参加全国成果展示。这些推荐的作品涉及中小学14个学科，较为全面地体现现 阶段我省中小学教师教育技术能力水平。

由全国中小学教师教育技术能力建设计划项目办公室举办的此次成果展示活动，旨在进一步推进中小学教师教育技术能力培训工作，促进项目培训 成果向教学实践应用转化，提高中小学教师教育技术应用能力。此次取得的成绩既是我省教师教育技术能力建设成果的展示，也是接下来常抓不懈和继 续加强教师教育技术能力工作的动力。各地级市获奖者、获奖单位也应把这次竞赛作为新的起点，在原有经验和成果的基础上，继续为我省中小学教师 培训工作做出新的贡献。

主办单位：广东省知识产权局
